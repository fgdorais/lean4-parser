    inductive Lake.Toml.KeyTy :
    Type

    The manner in which a TOML key was declared.

    • value : KeyTy

      A key declared via key = v.

    • stdTable : KeyTy

      A key declared via [key].

    • array : KeyTy

      A key declared via [[key]].

    • dottedPrefix : KeyTy

      A key declared via key.foo.

    • headerPrefix : KeyTy

      A key declared via [key.foo] or [[key.foo]].

                            Table

                            {a.b := [c.d := 0]}, {a.b := [c.e := 1]}
